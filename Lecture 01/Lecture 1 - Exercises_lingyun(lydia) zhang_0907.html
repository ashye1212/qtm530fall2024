(f) Generate "vec_x" from a chi-square
distribution with (df = 1, size = 1000)

  • Plot a histogram of each variable, with axis labels, and title
  • Create a new variable "vec_y" by the formula

$ y = e^x + 2x + 5$

  • Plot "vec_x" against "vec_y" and label the axes
In [ ]:
# Write your own code

import numpy as np

# Generate "vec_x" from a chi-square distribution with df=1 and size=1000
vec_x = np.random.chisquare(df=1, size=1000)

# Plot a histogram of "vec_x"
plt.hist(vec_x,color='pink')
plt.xlabel("Value")
plt.ylabel("Frequency")
plt.title("Histogram of vec_x")
plt.show()

# Create a new variable "vec_y" 
vec_y = np.exp(vec_x) + 2 * vec_x + 5

# Plot "vec_x" against "vec_y"
plt.scatter(vec_x, vec_y)
plt.xlabel("vec_x")
plt.ylabel("vec_y")
plt.title("Scatter Plot of vec_x vs vec_y")
plt.show()
